==================================================
RUNNING THE PROJECT
==================================================

1. Install Dependencies:
   cd MedSync/backend/services/auth
   npm install
   
   cd MedSync/backend/services/notification
   npm install
   
   cd MedSync/frontend
   npm install

2. Configure Environment:
   - All .env files are pre-configured in services
   - For email: Update EMAIL_USER and EMAIL_PASS in notification .env
   - For SMS: Update TWILIO_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM

3. Run with Docker Compose:
   docker-compose up --build

4. Run Locally:
   # Terminal 1 - Auth Service
   cd backend/services/auth
   npm run dev
   
   # Terminal 2 - Notification Service
   cd backend/services/notification
   npm start
   
   # Terminal 3 - Frontend
   cd frontend
   npm start

5. Access Admin Dashboard:
   http://localhost:3000
   Login: admin@medsync.com / your_password

==================================================
TROUBLESHOOTING
==================================================

Q: Auth service fails to connect to MongoDB
A: Ensure MONGO_URI is correct in .env file
   Default: mongodb://localhost:27017/medsync

Q: Notification emails not sending
A: Check EMAIL_USER and EMAIL_PASS in .env
   For Gmail, use app-specific passwords
   Enable "Less secure app access"

Q: Admin dashboard shows empty users list
A: Ensure auth service is running
   Check REACT_APP_API_BASE_URL in frontend .env
   Check browser console for API errors

Q: JWT token expired errors
A: Token expires in 7 days
   User needs to login again
   Clear localStorage and refresh page
